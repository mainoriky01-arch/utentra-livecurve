# Utentra LiveCurve

Sistema di selezione ventilatori industriali basato su dati Euroventilatori.

## Features

- 905 modelli ventilatori
- 45 serie, 5 categorie
- Curve caratteristiche con conversione logaritmica
- Precisione: errore <1.5%
